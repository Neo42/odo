## What is odo?

odo is a minimalist todo list app built with [Svelte](https://svelte.dev).

## How to use it?

- `Enter` -- Create a new todo
- `Tab` / `Shift + Tab` -- Navigate between todos
- `Delete` on Mac / `Backspace` on Windows -- Delete todo (content)
