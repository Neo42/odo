## What is odo?

odo is a minimalist todo list app built with [Svelte](https://svelte.dev). More features to come.

## How to use it?
- `enter` -- Create a new todo
- `tab` / `shift` + `tab` -- Navigate between todos
- `delete` on mac / `backspace` on win -- Delete todo (content)
